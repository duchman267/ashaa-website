# Ashaa Website

A minimalist website for Ashaa - Digital singer & songwriter.

## Features

- Clean, minimalist design
- Fast loading (static site)
- Responsive (mobile-friendly)
- Easy to edit

## Pages

- Home - Welcome & introduction
- About - Biography
- Music - Latest releases
- Contact - Get in touch

## How to Edit

1. Edit files in `content/` folder
2. Commit changes to GitHub
3. Site auto-deploys to Netlify

## Built With

- Hugo (Static Site Generator)
- Minimal custom theme
- Netlify (Hosting)
